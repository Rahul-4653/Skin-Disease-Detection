# Skin-Disease-Detection
Overview
This project is a deep learning-based image classification system designed to detect and classify multiple common skin diseases from input images.
The model uses Convolutional Neural Networks (CNNs) to analyze skin images and predict the most likely condition.
The system focuses on detecting closely related and commonly occurring skin conditions, making it useful for preliminary screening and awareness.
Diseases Covered

This model is trained to recognize and classify the following skin conditions:
Acne
Herpes
Eczema (Eksim)
Pityriasis Versicolor (Panu / Fungal Infection)
Rosacea
These conditions were selected because they are:
Visually distinguishable using image features
Common in real-world dermatology cases
Often confused due to similar symptoms
Model Architecture

The project uses a Convolutional Neural Network (CNN) that extracts key visual features such as:

Skin texture patterns
Color variations
Lesion shapes
Inflammation regions

These features are used to classify images into one of the supported disease categories.
